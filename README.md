# My_Robot_practice
## ROS2 Study & Project Log

이 저장소는 ROS2 핵심 개념(Topic, Service, Action 등) 학습 내용과 커스텀 패키지 구현 코드를 기록하는 공간입니다. 
단순한 코드 백업을 넘어, 빌드 과정에서 발생한 에러(Troubleshooting)와 해결 과정을 중점적으로 정리합니다.

## 🛠️ Environment
- **OS**: Ubuntu 22.04
- **ROS2 Version**: Humble
- **Language**: C++, Python
- **Build Tool**: Colcon, CMake
